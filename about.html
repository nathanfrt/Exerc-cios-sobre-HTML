HTML

Linguagem de marcação de hipertexto

A sigla HTML vem do inglês HyperText Markup Language ou em sua tradução Linguagem de Marcação de Hipertexto. Vale salientar que HTML não é uma linguagem de programação uma vez que seu objetivo é permitir que o browser (navegador) interprete através de marcações em seu conteúdo e renderize a tela para o usuário.

Toda página web é renderizada por uma linguagem HTML que garante a interação do usuário com o sistema. A linguagem possue diversos componentes com diferentes funcionalidades desde a informação à navegação. É através da estrutura HTML que o usuário consegue visualizar a interface da tela, os quais são diferenciados por tags e atributos.

As tags são a base ou estrutura de toda página HTML; através que determinamos as funções de cada conteúdo, como exemplo:

< p > São definidos os parágrados
< table > São definidos dados tabulados em linhas e colunas
< span > São elementos de estilização não semânticos
< div > São elementos agrupados não semânticos
< header > Referencia o cabeçalho
< footer > Referencia o rodapé da página
< input > Permite a entrada de dados como botão, checkbox, textos, senhas, etc.

Já os atributos são valores que definimos nas tags que definem como a tag irá comportar.
  • class: lista separada por espaço das classes do elemento;
  • id: define um identificador exclusivo que deve ser único em todo o documento;
  • style: contém regras de declarações CSS que se aplica no elemento;
  • title: representa os títulos do conteúdo relacionado ao elemento que tal atributo pertence.

    • Contudo, é de extrema valia uma boa estrutura HTML, pois é através dela que visualizaremos o conteúdo e toda interface de nosso website.
